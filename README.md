# HARAL - Halal Online Shop in Korea

[bozoraka.com](https://bozoraka.com/)과 같이 **쇼핑몰만** 제공하는 할랄 온라인 마켓입니다.

## 지원 언어

한국어 · English · Русский · O'zbekcha

## 페이지

| 경로 | 설명 |
|------|------|
| `/ko` | 메인 (상품 목록) |
| `/ko/shop/[id]` | 상품 상세 |
| `/ko/cart` | 장바구니 |
| `/ko/checkout` | 결제 |
| `/ko/account` | 계정 |

## 시작하기

```bash
cd haral-shop
npm install
npm run dev
```

http://localhost:3000
